# Paperflite Assignment

This repository contains solutions for three programming problems. Each question is organized in a separate folder with its own source code and explanation.

## Repository Structure

```
PAPERFLITE-ASSIGNMENT
│
├── Question1
│   ├── src
│   │   └── Solution.java
│   └── README.md
│
├── Question2
│   ├── src
│   │   └── Solution.java
│   └── README.md
│
├── Question3
│   ├── src
│   │   └── Solution.java
│   └── README.md
```

Each folder contains:
- Solution.java – Main implementation (includes main method)
- README.md – Problem explanation and sample run

## How to Clone the Repository

Open a terminal and run:

```
git clone https://github.com/lavenderlark/Paperflite-Assignment.git
```

Then navigate into the project directory:

```
cd paperflite-assignment
```

You can then access each question folder individually.
